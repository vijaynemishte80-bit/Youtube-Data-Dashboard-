# YouTube Data Dashboard using Streamlit

## Project Overview
This project is a *YouTube Data Dashboard* built with Python, Streamlit, and Plotly.  
It allows users to input a YouTube *Channel ID* and visualize important channel statistics such as subscribers, views, and videos. Users can also explore detailed analytics like top videos, likes vs views, comment distributions, and correlations.

## Features
- *Channel ID Input:* Enter any valid YouTube channel ID to fetch data.
- *Channel Statistics:* Displays subscribers, total views, and total videos.
- *Top 10 Videos Bar Chart:* Interactive chart showing most popular videos.
- *Likes vs Views Scatter Plot:* Bubble size represents number of comments.
- *Comments Histogram:* Shows the distribution of comments across videos.
- *Correlation Heatmap:* Analyzes relationships between views, likes, and comments.
- *CSV Export:* Download complete channel data for offline analysis.
- *Streamlit Interface:* Fully interactive and user-friendly dashboard accessible in the browser.

## Tools & Libraries
- Python
- Pandas
- Plotly
- Streamlit
- Google API (YouTube Data API v3)
